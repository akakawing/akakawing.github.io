## Git

Git配置设置

```bash
git config --global --list //查看配置设置
git config --global user.name YourUserName  //设置Username
git config --global user.email YourUserEmail  //设置UserEmail
```

Git 远程仓库连接查询

```bash
git remote -v
```

## Jekyll

更新bundle

```bash
bundle update
```

启动调试服务器

```bash
bundle exec jekyll serve
```
